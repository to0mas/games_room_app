<h2 align="left">Cílem tohoto projektu je vytvoření desktopové aplikace pro rezervaci školní herní místnosti (Games Room). Aplikace umožňuje žákům rezervovat místnost, upravovat a spravovat své rezervace. Admini mají přístup k funkcím pro správu uživatelů a rezervací.</h2>
