# games_room_app
Rezervační systém pro školní Games Room
Aplikace pro rezervaci školní místnosti, která bude sloužit k plánování a správě herní místnosti (games room).
Žáci budou mít možnost si místnost rezervovat podle dostupnosti, a spravovat své rezervace.
